# Easy Nextcord Bot Template 
Easy [nextcord](https://docs.nextcord.dev/en/stable/) discord bot that uses cogs

# How To Use
+ Clone repo
+ Install nextcord
```python
pip install nextcord
```
+ edit json/configuration.json
```json
  {
    "token": "bot token here",
    "prefix": "prefix here",
    "owner_name": "owner name here"
}
```
+ run main.py
```python
python main.py
```
