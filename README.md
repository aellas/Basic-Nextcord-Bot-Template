# Easy Nextcord Bot Template 
+ [Nextcord](https://docs.nextcord.dev/en/stable/) discord bot template
+ Easy setup 

# How To Use
+ Clone repo
+ Install nextcord
```python
pip install nextcord
```
+ edit json/configuration.json
```json
  {
    "token": "bot token here",
    "prefix": "prefix here",
    "owner_name": "owner name here"
}
```
+ run main.py
```python
python main.py
```
